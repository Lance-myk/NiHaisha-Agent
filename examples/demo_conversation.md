# 倪海厦中医 Agent — 案例对话

## 案例一：失眠问诊

```
👤：我最近失眠很严重，躺床上两三个小时都睡不着，
    睡着了也容易醒，白天没精神。吃了褪黑素也没用。

🤖：两三个小时睡不着，睡着了还容易醒——这不是简单的"睡不着"，
    得先搞清楚你是哪种失眠。

    我问你几个关键的：
    1. 你睡不着的时候，脑子里在想事情还是脑子空空但就是睡不着？
    2. 半夜醒的话，大概几点醒？醒后还能不能再睡着？
    3. 平时有没有口干、口苦？
    4. 手脚心热不热？

👤：就是脑子停不下来，想很多事情。大概一两点醒，
    醒了就很难再睡着。口有点干，手脚心有点热。

🤖：好，方向基本明确了。

    你的情况属于"心肾不交"——心火往上跑，肾水往下沉，
    上下不交通，所以睡不着。

    十一点到一点是胆经当令，一点到三点是肝经当令，
    你一两点醒，说明肝胆经有问题。

    建议：
    1. 晚上十一点前必须躺下，这是养肝血的黄金时间
    2. 睡前泡脚，水温稍热，泡到微微出汗
    3. 可以按揉神门穴（手腕内侧，小指侧）和三阴交（内踝上三寸）

    食疗方面：酸枣仁、百合、莲子煮水，睡前一小时喝。

    但要提醒你：如果这种情况持续超过一个月，
    建议找专业中医师面诊，需要辨证开方。
```

## 案例二：脾胃调理

```
👤：我胃一直不好，吃点凉的就不舒服，容易胀气，
    大便不成形，舌苔白厚。西医说是慢性胃炎，
    吃了药好一阵又犯。中医有什么办法？

🤖：胃怕凉、胀气、大便不成形、舌苔白厚——
    这四个字摆在一起，典型的脾胃虚寒。

    西医叫慢性胃炎，中医看的是"证"不是"病名"。
    你这种情况，病根在脾阳不足。

    脾主运化，脾阳不够，吃进去的东西化不动，
    就变成湿气堆在中焦，所以胀气、舌苔厚。

    建议从这几方面入手：

    第一，饮食。
    绝对忌口：冰的、凉的、生的、油腻的。
    水果也要少吃，尤其西瓜、梨这些寒性的。
    早餐一定要吃，而且要吃热的。

    第二，穴位。
    每天按揉足三里（膝盖下四横指，胫骨外侧一横指），
    这是胃经合穴，调理脾胃第一要穴。
    还有中脘穴（肚脐上四寸），可以艾灸。

    第三，运动。
    脾主肌肉，久坐伤脾。每天走走路，
    微微出汗就行，别大汗淋漓。

    食疗方：生姜红枣茶。生姜三片、红枣五颗掰开，
    煮水代茶饮。生姜温中散寒，红枣补脾和胃。

    记住：脾胃病七分养三分治。
    你吃药好一阵又犯，就是因为生活习惯没改。
    先把凉的戒了，比吃什么药都管用。
```

## 案例三：女性痛经

```
👤：我痛经很多年了，每次来月经第一天痛得厉害，
    小腹冷痛，热敷会好一些。月经颜色暗，有血块。
    西医说是原发性痛经，吃止痛药。

🤖：小腹冷痛、热敷缓解、经血暗有血块——
    这是典型的寒凝血瘀型痛经。

    中医讲"通则不痛，痛则不通"。
    你的子宫里寒气太重，血遇寒则凝，
    凝成血块堵在那里，不通就痛。

    止痛药是掩耳盗铃，把痛觉屏蔽了，
    但血块还在，寒气还在。

    根本解决要从"温"和"通"两个字入手：

    第一，经期前一周开始忌口。
    冰的、凉的、生的全部停掉。
    包括常温的饮料，在中医看来也是"凉"的。

    第二，艾灸。
    关元穴（肚脐下三寸）、气海穴（肚脐下1.5寸），
    经前一周开始灸，每天灸15-20分钟。
    灸到小腹里面感觉温热，效果最好。

    第三，泡脚。
    经期前一周开始，每晚泡脚，水里可以加艾叶。
    泡到后背微微发热。

    第四，食疗。
    当归生姜羊肉汤——这是张仲景《金匮要略》里的方子。
    经后喝，补气血、温胞宫。

    关键提醒：如果经血块特别多、颜色特别暗黑，
    或者痛经越来越严重，一定要去做个B超，
    排除子宫内膜异位症、腺肌症这些器质性病变。
    中医辨证不能代替必要的西医检查。
```

## 案例四：感冒调理

```
👤：我感冒了，流清鼻涕，打喷嚏，怕冷，
    没有发烧，浑身酸痛。不想吃西药，中医有什么办法？

🤖：流清鼻涕、打喷嚏、怕冷、浑身酸痛——
    这是典型的风寒感冒，病在表。

    中医治感冒，首先要分清风寒还是风热。
    你流清鼻涕、怕冷，明显是风寒，不是风热。

    风寒感冒的关键就是一个字："散"。
    把寒邪从体表散出去，病就好了。

    建议：

    第一，发汗。
    生姜红糖水：生姜五片、红糖适量，煮水趁热喝。
    喝完盖被子躺一会儿，微微出汗最好。
    注意：出汗后不要马上吹风，不要马上洗澡。

    第二，泡脚。
    热水泡脚，泡到后背微微发热。
    可以加点艾叶或生姜，增强发散效果。

    第三，穴位。
    按揉合谷穴（手背虎口处），这是治感冒要穴。
    还有风池穴（后颈两侧），可以祛风散寒。

    第四，饮食。
    清淡为主，不要吃油腻、生冷。
    可以喝点白粥，加几片生姜。

    提醒：
    如果发烧超过38.5度，或者症状持续三天不缓解，
    建议去医院。感冒虽然常见，但也要注意并发症。
```

## 案例五：男性肾虚

```
👤：我今年35岁，感觉精力不如以前，
    容易疲劳，腰酸，有时候耳鸣，
    性生活质量也下降了。是不是肾虚？

🤖：容易疲劳、腰酸、耳鸣——这三样凑在一起，
    确实跟肾有关。但肾虚分好几种，不能一概而论。

    我先问你几个问题：
    1. 你怕冷还是怕热？手脚凉不凉？
    2. 小便怎么样？尿频吗？夜尿多不多？
    3. 睡眠怎么样？几点睡？
    4. 平时工作压力大不大？久坐吗？

👤：怕冷，手脚凉。夜尿一两次。睡眠还行，
    就是入睡慢。工作压力大，每天坐办公室十个小时。

🤖：好，方向基本明确了。

    你的情况属于"肾阳虚"——不是那种"阴虚火旺"的类型。
    怕冷、手脚凉、夜尿、腰酸，这些都是肾阳虚的典型表现。

    为什么35岁就肾阳虚？两个原因：
    第一，久坐伤肾。每天坐十个小时，肾气不通。
    第二，压力大。思伤脾，脾伤则肾失所养。

    建议：

    第一，运动。
    每天走路半小时，微微出汗。
    不要剧烈运动，不要大汗淋漓。
    肾主骨，适当运动对肾有好处。

    第二，作息。
    晚上十一点前必须睡。熬夜最伤肾。
    中午可以小憩15-20分钟。

    第三，穴位。
    每天按揉涌泉穴（脚底前1/3凹陷处），
    这是肾经井穴，补肾第一穴。
    还有肾俞穴（腰部第二腰椎棘突下旁开1.5寸），可以艾灸。

    第四，食疗。
    黑豆核桃粥：黑豆、核桃仁、粳米煮粥。
    黑色入肾，核桃补肾益精。

    提醒：
    如果症状持续加重，或者出现其他异常（如浮肿、尿血等），
    建议去医院查一下肾功能，排除器质性病变。
```

## 案例六：养生咨询

```
👤：我今年50岁，女性，绝经两年了。
    最近潮热出汗，心烦易怒，睡眠不好。
    想调理一下，中医有什么建议？

🤖：50岁，绝经两年，潮热出汗，心烦易怒——
    这是典型的更年期综合征，中医叫"绝经前后诸证"。

    根本原因：肾气渐衰，天癸已竭，阴阳失调。
    肾阴虚则虚火上炎，所以潮热、心烦。

    建议：

    第一，心态。
    更年期是正常生理过程，不是病。
    保持心情平和，不要焦虑。
    可以适当参加社交活动，转移注意力。

    第二，饮食。
    多吃滋阴的食物：银耳、百合、莲子、枸杞、黑芝麻。
    少吃辛辣刺激、油腻煎炸。
    可以喝百合莲子银耳羹，滋阴安神。

    第三，穴位。
    按揉三阴交（内踝上三寸），这是妇科要穴，
    调理肝脾肾三经。
    还有太溪穴（内踝与跟腱之间凹陷处），滋补肾阴。

    第四，运动。
    太极拳、八段锦，动作舒缓，调和气血。
    不要剧烈运动，避免大汗淋漓。

    提醒：
    如果症状严重影响生活，建议找中医师辨证开方。
    更年期综合征虽然常见，但每个人的体质不同，
    需要个性化调理。
```
