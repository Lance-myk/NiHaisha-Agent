# 倪海厦中医顾问 — 病症专属问诊槽位表

> 以下槽位表基于倪海厦人纪系列（黄帝内经、伤寒论、金匮要略、神农本草经、针灸）整理。
> 每个病症有独立的问诊重点，顾问根据用户主诉自动匹配。

---

## 一、外感病类

### 1.1 太阳病（感冒/风寒/风热）

**问诊重点**：有汗无汗、怕冷怕热、寒热往来

| 槽位 | 必问/选问 | 问诊内容 |
|------|----------|----------|
| 主诉 | 必问 | 哪里不舒服？头痛？鼻塞？咳嗽？喉咙痛？ |
| 汗出 | 必问 | **有汗还是无汗？** 这是桂枝汤和麻黄汤的分界线 |
| 寒热 | 必问 | **怕冷还是怕热？** 恶寒发热是太阳病特征 |
| 二便 | 必问 | 大便正常吗？有没有便秘或拉肚子？ |
| 饮食 | 选问 | 胃口怎么样？想不想吃东西？ |
| 既往 | 选问 | 有没有老毛病？高血压、心脏病？ |
| 舌脉 | 选问 | 舌苔白还是黄？厚不厚？ |

**关键鉴别**：
- 有汗+恶风+脉浮缓 → 太阳中风（桂枝汤证）
- 无汗+恶寒+脉浮紧 → 太阳伤寒（麻黄汤证）
- 寒热往来+胸胁苦满 → 少阳病（小柴胡汤证）

---

### 1.2 温病/热病

**问诊重点**：发热程度、口渴、汗出特点

| 槽位 | 必问/选问 | 问诊内容 |
|------|----------|----------|
| 主诉 | 必问 | 发烧多久了？最高多少度？ |
| 汗出 | 必问 | **汗出后热退不退？** 还是持续发热？ |
| 口渴 | 必问 | **口渴不渴？** 想喝热水还是凉水？ |
| 寒热 | 必问 | 怕冷还是怕热？还是忽冷忽热？ |
| 二便 | 必问 | 大便干不干？小便黄不黄？ |
| 舌脉 | 选问 | 舌头红不红？舌苔黄不黄？ |

**关键鉴别**：
- 高热+口渴+汗出热不退 → 白虎汤证
- 寒热往来+口苦咽干 → 少阳证
- 午后发热+身热不扬 → 湿温

---

## 二、脾胃病类

### 2.1 脾胃虚寒

**问诊重点**：怕冷喜热、大便溏薄、舌苔白厚

| 槽位 | 必问/选问 | 问诊内容 |
|------|----------|----------|
| 主诉 | 必问 | 胃怎么不舒服？胀痛？反酸？还是吃了凉的就难受？ |
| 寒热 | 必问 | **怕冷还是怕热？** 喜欢喝热水还是凉水？ |
| 饮食 | 必问 | **胃口怎么样？** 吃凉的舒服还是不舒服？ |
| 二便 | 必问 | **大便怎么样？** 干还是稀？成形不成形？一天几次？ |
| 舌脉 | 必问 | **舌苔白不白？厚不厚？** 有没有齿痕？ |
| 睡眠 | 选问 | 几点睡？半夜醒不醒？ |
| 既往 | 选问 | 这个病多久了？怎么开始的？ |

**关键鉴别**：
- 胃怕凉+喜热饮+大便溏+苔白厚 → 脾胃虚寒
- 胃胀痛+反酸+大便干 → 胃热或胃阴虚
- 胃胀+嗳气+情绪差 → 肝胃不和

---

### 2.2 便秘

**问诊重点**：大便干结程度、寒热虚实

| 槽位 | 必问/选问 | 问诊内容 |
|------|----------|----------|
| 主诉 | 必问 | 几天一次大便？干不干？排便费力吗？ |
| 寒热 | 必问 | **怕冷还是怕热？** 手脚凉不凉？ |
| 饮食 | 必问 | 胃口怎么样？吃得多不多？ |
| 二便 | 必问 | 小便怎么样？颜色黄不黄？ |
| 舌脉 | 选问 | 舌头红不红？舌苔厚不厚？ |
| 既往 | 选问 | 便秘多久了？是突然的还是长期的？ |

**关键鉴别**：
- 大便干+怕冷+手脚凉 → 脾肾阳虚（冷秘）
- 大便干+口干+舌红 → 阴虚肠燥
- 大便不干但排不出+腹胀 → 气虚便秘

## 三、妇科病类

### 3.1 痛经

**问诊重点**：小腹冷热、血块、月经颜色

| 槽位 | 必问/选问 | 问诊内容 |
|------|----------|----------|
| 主诉 | 必问 | 小腹冷还是热？热敷能不能缓解？ |
| 月经 | 必问 | **月经颜色暗不暗？有没有血块？** |
| 寒热 | 必问 | **平时怕冷还是怕热？** 手脚凉不凉？ |
| 舌脉 | 必问 | **舌苔白不白？** 舌头底下青筋粗不粗？ |
| 既往 | 必问 | **从初潮就痛，还是后来才痛的？** |
| 二便 | 选问 | 大便怎么样？干还是稀？ |
| 睡眠 | 选问 | 几点睡？半夜醒不醒？ |

**关键鉴别**：
- 小腹冷痛+热敷缓解+经血暗有血块 → 寒凝血瘀
- 小腹灼热+经血鲜红+口干 → 血热
- 小腹胀痛+情绪差+乳房胀 → 肝郁气滞

---

### 3.2 更年期综合征

**问诊重点**：潮热出汗特点、寒热真假、睡眠

| 槽位 | 必问/选问 | 问诊内容 |
|------|----------|----------|
| 主诉 | 必问 | 潮热出汗是白天多还是晚上多？出汗后怕冷吗？ |
| 寒热 | 必问 | **平时怕冷还是怕热？** 手脚心热不热？ |
| 二便 | 必问 | 大便干不干？小便频不频？ |
| 饮食 | 必问 | 口干不干？想不想喝水？喜欢喝热的还是凉的？ |
| 舌脉 | 必问 | **舌头红不红？舌苔少不少？** |
| 睡眠 | 选问 | 睡不着还是容易醒？大概几点醒？ |
| 既往 | 选问 | 绝经多久了？之前月经正常吗？ |

**关键鉴别**：
- 潮热+手脚心热+口干+舌红少苔 → 肝肾阴虚
- 潮热+怕冷+乏力 → 阴阳两虚
- 心烦+失眠+易怒 → 心肝火旺

## 四、肾病类

### 4.1 肾虚（男性）

**问诊重点**：怕冷怕热、二便、性功能

| 槽位 | 必问/选问 | 问诊内容 |
|------|----------|----------|
| 主诉 | 必问 | 腰酸还是腿软？精力差到什么程度？ |
| 寒热 | 必问 | **怕冷还是怕热？** 手心脚心热不热？ |
| 二便 | 必问 | **夜尿多不多？** 小便清长还是黄少？ |
| 睡眠 | 必问 | 几点睡？半夜醒不醒？醒了还能不能睡着？ |
| 舌脉 | 选问 | 舌头红不红？舌苔厚不厚？ |
| 既往 | 选问 | 这个问题多久了？突然的还是慢慢发展的？ |

**关键鉴别**：
- 怕冷+手脚凉+夜尿多+小便清长 → 肾阳虚
- 怕热+手脚心热+盗汗+口干 → 肾阴虚
- 腰酸+乏力+性功能下降+苔白厚 → 脾肾阳虚

## 五、心系病类

### 5.1 失眠

**问诊重点**：入睡困难还是易醒、寒热、二便

| 槽位 | 必问/选问 | 问诊内容 |
|------|----------|----------|
| 主诉 | 必问 | 是睡不着，还是容易醒？大概几点醒？ |
| 寒热 | 必问 | **手脚心热不热？** 晚上有没有口干口苦？ |
| 二便 | 必问 | 大便怎么样？干不干？ |
| 舌脉 | 选问 | 舌尖红不红？舌苔少不少？ |
| 既往 | 选问 | 最近有没有烦心事？压力大不大？ |
| 饮食 | 选问 | 下午以后喝咖啡、浓茶多不多？ |

**关键鉴别**：
- 睡不着+心烦+手脚心热+口干 → 心肾不交
- 容易醒+多梦+乏力 → 心脾两虚
- 凌晨1-3点醒+易怒 → 肝火旺

| 槽位 | 必问/选问 | 问诊内容 |
|------|----------|----------|
| 主诉 | 必问 | 胁肋胀痛还是刺痛？跟情绪有没有关系？ |
| 睡眠 | 必问 | **几点醒？** 凌晨1-3点醒不醒？ |

| 二便 | 选问 | 大便干不干？小便黄不黄？ |
| 既往 | 选问 | 最近有没有生气、忧虑、压力大？ |

**关键鉴别**：
- 胁痛+情绪差+乳房胀 → 肝郁气滞
- 胁痛+口苦+目赤+易怒 → 肝火旺
- 凌晨1-3点醒+多梦+爪甲苍白 → 肝血虚

| 槽位 | 必问/选问 | 问诊内容 |
|------|----------|----------|

- 皮肤干燥+痒+怕冷 → 血虚风燥

## 十、通用问诊槽位（所有病症适用）

| 槽位 | 用途 | 问诊内容 |
|------|------|----------|
| **年龄性别** | 判断生理阶段 | 多大年纪？男的女的？（女性问月经） |
| **既往病史** | 排除禁忌 | 有没有慢性病？在吃什么药？ |
| **情志变化** | 判断肝郁 | 最近有没有生气、忧虑、压力大？ |
| **生活起居** | 判断习惯 | 几点睡？睡得好不好？工作累不累？ |

## 问诊原则总结

1. **先辨病再问诊**：根据用户主诉，快速匹配到对应病症类别
2. **必问优先**：每个病症有2-3个必问槽位，先问关键的
3. **选问补充**：信息够了就分析，不要没话找话
4. **灵活穿插**：不要列表式提问，要像聊天一样自然
5. **关键鉴别**：通过1-2个关键问题区分不同证型
